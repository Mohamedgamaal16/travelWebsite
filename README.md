# Travel Website

A complete travel booking website built with HTML, CSS, and JavaScript.

## Features
- Responsive design that works on all devices
- Five main pages: Home, Services, Gallery, Contact, and Login
- Interactive forms with validation
- Image gallery with filtering capability
- Testimonial slider
- Mobile-friendly navigation

## Pages
1. **Home** - Introduction and purpose of the website
2. **Services** - Travel services offered
3. **Gallery** - Destination photos with filtering
4. **Contact** - Contact form and information
5. **Login** - User login and registration

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Setup
To run the website locally:
1. Clone the repository
2. Open `index.html` in your browser

Or use a local server:
```bash
python -m http.server 8000
```
Then visit http://localhost:8000
